# Security Policy

## Reporting a Vulnerability

BlackCortex Lite is a closed-source indie project maintained by a single developer.

If you discover a security vulnerability, please **do not open a public GitHub issue**.

Report it privately via email:

📧 **hello@black-cortex.com**

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

We will respond as soon as possible. There is no formal bounty program, but responsible disclosure is appreciated and will be acknowledged.

## Scope

This repository contains only documentation and release binaries — no source code is published here. Security reports should relate to the **BlackCortex Lite application behavior**, not to this repository itself.

## Antivirus False Positives

If your antivirus flags `BlackCortex-Lite.exe`, this is a known false positive caused by PyInstaller packaging. See [INSTALL.md](INSTALL.md) for details and VirusTotal results.
